# Algoflix PHP

Projeto acadêmico desenvolvido em PHP utilizando MVC, PDO e MySQL.

## Tecnologias

- PHP 8
- MySQL
- XAMPP
- HTML5
- CSS3

## Funcionalidades

- Login
- Cadastro
- Favoritos
- Avaliações
- Comentários
- Histórico
- Dashboard Administrativo
- Proteção CSRF

## Instalação

1. Clone o projeto

```bash
git clone https://github.com/seuusuario/algoflix_php.git
```

2. Crie o banco

```sql
CREATE DATABASE algoflix_db;
```

3. Execute:

```text
sql/estrutura.sql
```

4. Execute:

```text
sql/dados_teste.sql
```

5. Inicie Apache e MySQL no XAMPP

6. Acesse:

```text
http://localhost/algoflix_php
```
